# 递归

在方法中调用本方法

特点：

+ 自己调用自己
+ 有终止条件

## 斐波那契数列

```java
//获取斐波那契数列第i个值
public static int fibonacciItem(int i){
    if(i<1){
        throw new IllegalArgumentsException("参数为非零正整数");
    }
    if(i==1 || i==2){
        return 1;
    }else{
        return fibonacciItem(i-1)+fibonacciItem(i-2);
    }
}

//打印前n个斐波那契数列
// 迭代法
public static void fibonacciSeq(int n){
	int pre = 1;
    int now = 1;
    int mid;
    for(int i=1; i<=n; i++){
        if(i<=2){
			sout(1+" ")
        }else{
        	mid = pre + now;
            sout(mid+" ");
            pre = now;
            now = mid;
        }
    }
}
```
