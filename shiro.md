# 1 认识shiro

shiro是Java的一个权限管理安全框架，可以用它完成认证、授权、加密、会话管理、与web集成、缓存等

## 1.1 功能简介

+ 核心功能
  1. 认证
  2. 授权
  3. 会话管理
  4. 加密
+ 支持特性
  1. web支持
  2. 缓存
  3. 一致性(多线程情况下)
  4. 测试
  5. "Run As"
  6. Remember Me
